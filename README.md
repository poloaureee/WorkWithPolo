# Static Portfolio (HTML/CSS/JS)

Open `Porfolio/index.html` in your browser (or deploy this `Porfolio/` folder to Netlify/Vercel/GitHub Pages).

## Quick edits

- **Content**: update the arrays in `Porfolio/script.js` (`content.services`, `content.experiences`, `content.projects`)
- **Links**: replace any remaining placeholder `#` links in `Porfolio/script.js` (project demo URLs)
